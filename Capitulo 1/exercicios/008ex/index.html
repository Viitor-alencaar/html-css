Texto marcado

Tambem podemos criar um texto marcado usando a tag mark. ex: "Serei um grande programador".

obs:O html não possui a forma de marcar com outra cor se não amarelo. Para isso, precisamos usar o css e colocar "style=backgound-color;(e a cor que voce desejar)

Texto grande e texto pequeno

Para criar um texto maior dos outros, digitamos no ctrl+shift p, digitamos abb e depois colcomos a palagra "big"
EX:Eu e minha casa serviremos ao senhor
Para fazer uma pequena, faremos o mesmo processo e depois digitamos "small1"
EX: Esse é um exemplo de palavras pequenas

texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido mas não considerado

Para que esse texto seja considerado como excluido, podemos envelopar ele e colcoar a tag "del"
Ex: Eu fiz algo especial


Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase para indicar que ele foi adicionado depois adicionando a tag "ins"

ex: eu gostaria de aprender mais


Texto sobrescrito

Nessa parte, temos o conhecimento de que um numero por exemplo, fica em cima, igual no 2 elevado ao quadrado. Basta enveloparmos e colocarmos a tecla "sup"


Ex: x20+3

Texto subescrito

Nesse texto, podemos colocar a tecla em baixo, igual na sigla "H2O".
Basta enveloparmos e colocar a tecla "sub"


Ex: H2O
