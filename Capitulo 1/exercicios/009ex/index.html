Trabalhando com listas

Listas ordenadas

Para formar uma lista, basta adicionar "Ol" e depos "li"

  1. Acordar as 5am
  2. Escovar os dentes
  3. Ir pra academia 5:20
  4. Tomar café 7:30
  5. ir pra faculdade 8:00

Listas não ordenadas

São listas que não fazem sentido. não tem ordem. apenas colocar as letras "ul"

  • Acordar as 8:00
  • Ir trabalhar 9:00
  • tirar intervalo

Juntando listas

Podemos criar uma lista dentro de outro lista colocando "ol" dentro de outro "ol". Ex:

Coisas que devo estudar

  1. Direito penal
    1. Art 129
    2. Art 130
    3. Lei de drogas
  2. Direito civil
    • Direito de sucessões
    • Direito de familia
    • Contratos

Lista de definições

Podemos fazer tambem uma lista de definições. Para formar essa lista, pedemos usar as seguintes coisas:

  1. "dl" para abrir a lista
  2. "dt" para indicar o termo
  3. "dd" para indicar a Descrição
Ex:

HTML
Linguagem de marcação para a criação do conteúdo de um site
CSS
Linguagem de marcação para a criação de um desing de um site
Javascript
Linguagem de programação para a criação de interatividade de um site
