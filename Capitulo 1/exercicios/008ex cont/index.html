Outras formatações

Codigo fonte

Podemos utilizar a tag "code" para monoespaçar as palavras. Deixar todas em uma mesma largura. Basta envelopar o que voce quer espaçar e depois apertar "code"

Ex: Essa frase esta com algumas diferenças de espaçamento

Podemos utilizar tambem o code "pre" que da o espaçamento adequado

Ex: 

        
            eu quero estar melhor no futuro

Citações simples

Para fazer uma citação, devo envelopar e apertar o "q"

Ex: Como diria um pai de um amigo: O computador é um burro muito rapido

Citações completas

Para fazer uma citação mais completa, podemos utilizar a tag "bloclquote" e fazer uma citação mais completa.

Por exemplo: Segundo o autor Yuval noah Harari em seu livro sapiaens:

A capacidade de criar uma realidade imaginada a partir de palavras permitiu que grandes quantidades de estranhos cooperassem de forma eficaz.

Abreviações

Podemos usar a tag "abbr" em abreviações para dizer o qu essa abreviação significa

Ex: Estpu estudando HTML e CSS
